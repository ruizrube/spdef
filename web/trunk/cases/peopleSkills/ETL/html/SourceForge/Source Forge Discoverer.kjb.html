Job Entries

Source Forge Parameters Generator

Esta transformación se encarga de crear una URI correcta para posteriormente pedir a SourceForge, a través de su canal RSS, los nombres de todos sus proyectos. Cada petición RSS devuelve 100 nombres y la URI que debemos utilizar para obtener los 100 siguientes. Por ello se utiliza un fichero temporal (debe especificarse) donde se almacena próxima URI a llamar.

Source Forge Register

En esta transformación se guardan en la base de datos Abreforjas los nombres de los proyectos leídos en el último fichero.

Source Forge URI Holder

Este paso obtiene la siguiente URI a llamar y la guarda en una variable y en un fichero, por si el proceso se parara, poderlo continuar y no tener que empezar de 0

RSS Reader

Esta transformación hace un petición HTTP a la URI que recibe y obtiene de ella un documento XML que parsea para obtener los nombres de los proyectos. También parsea y guarda la siguiente URI
