Job Entries

Download Project Data

Esta transformación descarga los metadatos con la informacion general de cada proyecto

Update DB

Este paso sirve para actualizar el estado de los proyectos, indicando que sus metadatos ya han sido descargados. También añade en la base de datos la fecha de la última descarga de dichos metadatos para tener así una referencia

Download SF Tickets

Los tickets de SourceForge deben ser tratados de forma especial, ya que la información detallada de cada uno se obtiene a partir de una URI que utliza el código de cada ticket, para lo cual es necesario haber descargado y leído antes el ficher que contiene la información general

Set COUNTER

Este contador sirve para cuando hay más de un fichero de un determinado tipo por proyecto (tickets1, tickets2, etc)
